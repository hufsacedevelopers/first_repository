# 🚀 장애인 기업 친화도 평가 플랫폼

> 취업 정보를 넘어서, 기업을 데이터로 평가하는 플랫폼

---

## 📌 프로젝트 소개

장애인을 위한 **기업 장애 친화도 평가 플랫폼**입니다.  
단순 채용 정보 제공이 아닌, 공공데이터를 기반으로 기업의 장애 친화도를 정량화하여  
사용자가 **데이터 기반으로 직장을 선택할 수 있도록 돕는 서비스**를 목표로 합니다.

---

## 🎯 기획 배경

기존 취업 서비스는 대부분 채용 정보 제공에 그치며,  
실제 근무 환경이나 지속 가능성에 대한 정보는 부족합니다.

본 프로젝트는  
👉 **“갈 수 있는 직장”이 아닌**  
👉 **“오래 다닐 수 있는 직장”을 선택할 수 있도록**  
새로운 기준을 제시합니다.

---

## 💡 핵심 기능

- 📊 **기업 장애 친화도 점수 제공**  
  - 장애인 고용률, 근속률 등 데이터 기반 산출  

- 🆚 **기업 비교 기능**  
  - 여러 기업을 한눈에 비교 가능  

- 🗺️ **지도 기반 시각화**  
  - 지역별 기업 정보 및 접근성 확인  

- 📈 **설명 가능한 추천**  
  - 점수 산정 근거 제공  

---

## 🧠 차별화 포인트

- 단순 정보 제공 ❌  
- 데이터 기반 의사결정 지원 ✅  

> 기업을 선택하는 기준을 “감”이 아닌 **데이터**로 바꿉니다.
