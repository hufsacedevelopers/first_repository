# 기획 배경

## 문제 정의
- 장애인 및 청년 구직자는 기업의 실제 근무 환경 정보를 얻기 어렵습니다.
- 기업은 장애인 채용 시 활용 가능한 지원금과 제도를 충분히 인지하지 못합니다.
- 기존 채용 플랫폼은 공고 중심으로 정보가 제공되어 의사결정을 돕기 어렵습니다.

## MVP 목표
- 기업의 장애 친화도 점수를 확인할 수 있는 기본 화면 제공
- 사용자 관점의 추천 일자리 목록 제공
- 기업 관점의 지원금/혜택 정보를 API로 제공
- 실제 공공데이터 API 연동 전, mock 데이터로 기획 검증

## 확장 방향
- 공공데이터 API 연동 자동화 및 데이터 갱신 주기 설계
- PostgreSQL 도입으로 기업/일자리/추천 이력 영속화
- AI 추천 로직 고도화(사용자 프로필, 선호, 지역, 접근성 기준 반영)
