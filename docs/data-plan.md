# 데이터 활용 계획

## 현재 단계 (Mock 기반)
- `data/companies.json`: 기업 장애 친화도 계산 결과를 표현하는 샘플 데이터
- `data/jobs.json`: 사용자 추천 일자리 예시 데이터
- `data/supports.json`: 고용주 대상 지원금/혜택 예시 데이터

## 추후 공공데이터 연동 대상
- 장애인 고용률/고용현황 관련 공개 데이터
- 지역별 일자리 정보(워크넷 등)
- 장애인 고용 지원금/정책 안내 데이터

## 데이터 파이프라인 초안
1. 외부 API 수집 (배치/스케줄러)
2. 표준 스키마로 정규화
3. 장애 친화도 점수 계산
4. 백엔드 API 제공 및 프론트 노출

## 스키마 확장 메모
- 기업: 산업군, 기업규모, 최근 채용공고 수, 접근성 설비 점검 항목
- 일자리: 직무 요구역량, 경력요건, 근무시간 유연성
- 지원금: 신청 조건, 신청 기한, 신청 링크
